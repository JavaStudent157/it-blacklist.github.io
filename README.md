## 介绍

诞生于石家庄的IT黑企小程序，记录有不良行为的黑心企业，目前黑名单数据仅限于石家庄。

## 安装

小程序主要使用的技术有：Taro、Taro UI、Dva.js。

```bash
# yarn
git clone https://github.com/liujiayii/sjz_it_blacklist.git
cd client
yarn
```
