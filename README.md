+ 诞生于石家庄的IT黑企小程序，记录有不良行为的黑心企业。
+ 小程序主要使用的技术有：uni-app uni-cloud。

## 开始开发

小程序主要使用的技术有：Remax。
```bash
$ git clone https://github.com/liujiayii/sjz_it_blacklist.git
# 可能clone的项目不能正常运行，需要有一定解决问题的能力，或者直接问我
```
安装依赖

```bash
$ yarn install
```

开始构建

```bash
$ yarn run serve
```

## 发布

```bash
$ yarn run build
```
